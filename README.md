## Description

## venv
Generate your virtual environment

```bash
python -m venv env
```

Start it
```bash
# Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\env\Scripts\activate
```

## Installation
```bash
pip install -r requirements.txt
```
